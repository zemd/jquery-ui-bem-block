jquery-ui-bem-block
===================

Base jquery-ui widget to manipulate classes like in bem methodology


 * https://github.com/bem/
 * http://api.jqueryui.com/jQuery.widget/

 
Usage
=====

Simple inherit from it: 

```
$.widget('ns.myWidget', $.opesho.block, {});
```
